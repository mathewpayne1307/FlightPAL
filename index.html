Frequently Asked Questions

How are flights logged?

FlightPAL uses user-initiated start and stop actions to record block times, gps location to identify airfield, and configurable defaults for other flight details.

What is the Flight Monitoring?

When enabled, Flight Monitoring uses the same functions as normal logging for airfield and other default settings, but Off block times and On block times are determined by the app. Using GPS and Motion data, we are able to determine when the aircraft transitions from Parked > Taxi > Airborne > (Any Touch and Go's) > Taxi > Parked > Completed. The user has control on if flights are logged Off block to On block, or Takeoff to Landing.

Can I edit my flight entries?

Yes. All logged flights can be reviewed and edited within the app if any information needs adjusting.

Where does weather data come from?

METAR weather data is provided by the NOAA Aviation Weather Center and is displayed for informational purposes.

What is FlightPALs?

FlightPALs is a way of staying connected to your fellow pilots. Currently implemented Profile features include Achievements and Records, and an interactive map view showing all your logged flights. Soon to be implemented features of FlightPALs will include: Adding Friends Ability to see friends summaries, Records and Achievements Flight XP System, with profile level which unlock unique rewards.
