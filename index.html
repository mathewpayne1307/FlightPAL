Frequently Asked Questions

How are flights logged?

FlightPAL uses user-initiated Start and Stop actions to record block times, plus GPS location to help identify airfields, alongside configurable defaults for other flight details.

What is Flight Monitoring?

When enabled, Flight Monitoring uses GPS and motion data to help determine transitions like Parked → Taxi → Airborne → Taxi → Parked. You can choose whether flights are logged Off block → On block, or Takeoff → Landing.

Can I edit my flight entries?

Yes — logged flights can be reviewed and edited in the app if anything needs adjusting.

Where does weather data come from?

METAR weather data is provided by the NOAA Aviation Weather Center and shown for informational purposes.

What is Flight PALs?

Flight PALs is the social side of FlightPAL. Current profile features include achievements and records, with additional friend and profile interaction features planned.
